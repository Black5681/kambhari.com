## Local Development

To run this project locally:

1. Clone the repository
```bash
git clone https://github.com/Black5681/kambhari.com.git
cd kambhari.com
```

2. Install dependencies
```bash
npm install
```

3. Start the server
```bash
node server.js
```

4. Open http://localhost:3000 in your browser
